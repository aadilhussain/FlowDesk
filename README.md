# FlowDesk

FlowDesk is a multi-tenant SaaS workflow and task management platform built with Laravel and Vue.js.

## Tech Stack

- Laravel 13
- Vue.js
- Sanctum Authentication
- MySQL
- REST APIs

## Features Implemented

- API Authentication
- Sanctum Token System
- Register/Login APIs
- Protected Routes
- API Versioning

## Setup

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
